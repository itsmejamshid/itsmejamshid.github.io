Corne Keyboard Build

December 15, 2024
Hardware Soldering 3D Printing Electronics
GitHub Gallery

Project Overview

A custom split mechanical keyboard built from scratch using the Corne v3 design. This project involved component selection, PCB soldering, firmware programming, and 3D printing a custom case.

Specifications

  • Layout: 42-key split ortholinear
  • Switches: Akko Penguin Silent (tactile)
  • Keycaps: DSA profile blank black
  • Connectivity: Wireless (Bluetooth)
  • Firmware: ZMK
  • Case: 3D printed PLA

Motivation

In early October, I decided I wanted to build a split keyboard. I first encountered such keyboards on Shahzod's telegram channel, but the real inspiration came from a YouTube video where someone built a keyboard completely from scratch.

After watching more similar videos and discovering amazing communities on Reddit (r/crkbd, r/ErgoMechKeyboards), seeing different people's keyboards every day, I finally decided to build my own.

Design Choices

First, I needed to choose exactly what kind of keyboard to build. There are many types on the market, but some are more popular than others, like the Corne, Lily58, or Kinesis.

Lily58

58-key layout with number row

Kinesis

Full-featured ergonomic layout

Corne v3 ✓

42-key minimalist split layout

The Lily58 and Kinesis seemed to have too many keys for me, and I wanted something new, so I chose to build the "Corne v3".

Components & Sourcing

PCB Kit

Corne V3 Mx Choc Crkbd Split Keyboard kit from Taobao (~$30)

  • 2x PCBs with RGB underglow
  • 2x Pro Micro compatible controllers
  • Diodes and resistors
  • OLED displays

Switches

Akko Penguin Silent switches (~$15)

  • Tactile, silent operation
  • 62g actuation force
  • Hot-swappable MX compatible

Keycaps

DSA blank black keycaps x40 (~$7)

  • Uniform profile
  • Minimalist aesthetic
  • PBT plastic construction

Sourcing Strategy: After not finding kits locally in Uzbekistan, I searched international platforms. While AliExpress seemed more reliable, Taobao was significantly cheaper (almost half the price), so I ordered the main kit from Taobao and accessories from AliExpress.

Build Process

1

SMD Component Soldering

Started with the most challenging part - soldering tiny SMD diodes and resistors. Initially intimidated by the component size, I got help from a relative experienced in electronics. This turned out to be the most interesting part of the build.

Challenge: Never soldered before, components extremely small
2

Microcontroller Installation

Attached the Pro Micro controllers and added batteries for wireless operation. Unfortunately burned out one controller during the process and had to wait 15 days for a replacement.

Setback: Burned out microcontroller, 15-day delay for replacement
3

Firmware Flashing

The moment of truth - flashing ZMK firmware to test the soldering work. Initial attempt resulted in only 4 working keys out of 42, causing near panic. The issue was using the wrong firmware version.

Crisis: Only 4 keys working initially - wrong firmware version
4

3D Printing the Case

Downloaded case model from printables.com but encountered scaling issues. The printer was somehow reducing dimensions, requiring multiple test prints and manual scaling adjustments.

Frustration: 5 failed prints due to scaling issues

Technical Challenges & Solutions

SMD Soldering Precision

Problem: Never soldered before, components extremely small

Solution: Partnered with experienced relative, learned through hands-on practice

Firmware Compatibility

Problem: Only 4/42 keys working after initial flash

Solution: Research on Reddit revealed wrong firmware version, reflashed with correct ZMK build

3D Print Scaling

Problem: Printed cases consistently too small despite correct model dimensions

Solution: Manual scaling compensation (131-140%) after multiple test prints

Component Failure

Problem: Burned out microcontroller during assembly

Solution: Ordered replacement, improved handling procedures for sensitive components

Future Improvements

OLED Screen Integration

The kit included OLED displays, but ZMK firmware doesn't support them yet. Planning to migrate to QMK for screen functionality showing layer status and typing stats.

RGB LED Debugging

Soldered RGB LEDs but only one is functioning. Need to troubleshoot the LED chain and potentially re-solder connections.

Tenting System

Planning to add adjustable tenting legs for better ergonomics. Considering both 3D printed and commercial solutions for optimal wrist positioning.

Lessons Learned

Start Small with Hardware

SMD soldering requires practice. Consider starting with through-hole components or getting help for your first SMD project.

Firmware Research is Critical

Spend time understanding firmware options before building. QMK vs ZMK have different feature sets and compatibility requirements.

3D Printing Calibration

Test print scaling and tolerances before committing to long prints. Printer calibration saves time and materials.

Community Resources

Reddit communities like r/crkbd and r/ErgoMechKeyboards are invaluable for troubleshooting and inspiration.
